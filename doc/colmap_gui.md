# 如何用 COLMAP GUI 查看 sparse/0
要在 COLMAP GUI 中查看 sparse/0 的重建结果，请按照以下步骤操作：

2. 使用系統包管理器安裝 (最快)
Ubuntu 上可以直接安裝完整的 COLMAP（包含 GUI）：
```bash
sudo apt update
sudo apt install colmap
```

## 1. 啟動 GUI
在終端機輸入以下命令：
```bash
colmap gui
```
 

## 2. 導入模型 (Import Model)
進入界面後，點擊頂部菜單欄：  
`File > Import model`  

在彈出的文件選擇框中，導航到你的路徑：  
`~/Documents/jobs/task/task-blender/task03ai--img-modeling/dat/colmap_workspace/sparse/0`  

**關鍵點**：選擇 `0` 這個文件夾本身，然後點擊「選擇/打開」。  
**注意**：不要去選裡面的具體 `.bin` 文件，只需選中包含它們的文件夾。

## 3. 視圖操作
一旦模型加載，你應該能看到紅色的相機位姿和白色的稀疏點雲：
- 旋轉：左鍵點擊並拖動。
- 縮放：鼠標滾輪。
- 平移：右鍵點擊並拖動。
- 調整大小：如果相機或點太小，按鍵盤上的 `+` 或 `-` 鍵來縮放相機圖標和點的大小。

## 4. 檢查數據 (可選)
如果你想看具體有多少張圖被成功重建進去了：  
點擊菜單 `Extras > Model statistics`，它會顯示當前模型（即分區 0）包含的相機數、圖像數和 3D 點數。

### 提示：
如果 sparse/0 只有一部分場景，你可以重複上述步驟導入 sparse/1 或 sparse/2。如果發現主體被打碎在多個文件夾裡，說明之前的 **抽取圖片過稀** 或 **特徵點匹配不足** 導致了場景斷裂。
